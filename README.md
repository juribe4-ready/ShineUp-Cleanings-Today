# 🧹 ShineUP Cleaning App

Mobile operations app for cleaning staff at ShineUP Cleaning Services.

## 🏗️ Stack
- **Frontend:** Zite (React/TypeScript)
- **Backend:** Zite Endpoints
- **Database:** Airtable
- **Auth:** Gmail via Zite Auth SDK
- **Storage:** Google Drive

## 📱 Apps
1. **Staff Mobile App** - Daily cleaning operations
2. **Admin Dashboard** - Timeline view (WIP)

## 📁 Structure
- `/src/components/` - React components
- `/src/endpoints/` - Backend functions
- `/docs/` - Documentation

## 🎨 Design
- Color: Teal (#00BCD4)
- Font: Poppins
- Mobile-first
